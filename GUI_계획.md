# 블로그 자동화 GUI 프로그램 계획

## 메뉴 구성
- 글 작성 (키워드 입력 → 글 생성 → 플랫폼 선택 → 발행)
- 키워드 (추천 키워드 목록, 베스트 6 확인)
- 발행 현황 (오늘 발행된 글 목록, 예약 발행 현황)
- API 설정 (Claude, Naver, Gemini, Cloudinary 등 키 입력)
- 라이선스 (라이선스 키 인증)

## 기능 목록
- 키워드 자동 수집 (네이버 뉴스 기반)
- Claude AI 블로그 글 자동 생성
- Gemini 이미지 자동 생성 + 썸네일
- 블로그스팟 자동 발행
- 네이버 반자동 발행 (글 생성 후 복붙)
- 워드프레스 자동 발행 (추후)
- 일일 키워드 이메일 발송 (오전 5시)
- 베스트 6 자동 선정 + 4시간 간격 예약 발행
- 발행 예정 시간 확인 (0시/4시/8시/12시/16시/20시)
- 수동 발행 / 자동 발행 선택 가능
- 진행 상황 실시간 표시 (프로그레스바)

## 베스트 6 선정 기준
- 숏테일 2개 + 롱테일 2개 + 행동형 1개 + 정보형 1개
- 검색량 잠재력, 경쟁도, 광고 단가, 카테고리 다양성 점수제

## 판매용 패키징
- .exe 파일로 빌드 (PyInstaller)
- 라이선스 키 인증 시스템
- 구매자가 자신의 API 키 직접 입력하는 설정 화면
- GUI: tkinter 또는 PyQt

## 타겟 사용자
- 30~80세대 타겟 블로거
- 코딩 경험 없는 일반인도 사용 가능하게
