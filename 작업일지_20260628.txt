  - 원인: token.pickle 만료
  - 해결: generate_token.py로 재발급 → GitHub Secret 업데이트

[오류2] 중복 발행 (손담비 글 2~3개)
  - 원인: auto_publish.py에서 발행 후 JSON 저장 안 됨 → 항상 published:false
  - 해결: json.dump 코드 추가

[오류3] 네이버용 이메일 미발송
  - 원인: send_naver_email() 호출 코드 누락
  - 해결: 코드 복구

[오류4] git ORIG_HEAD.lock 충돌
  - 원인: 클로드 샌드박스에서 git 조작 시 lock 파일 생성
  - 해결: del .git\ORIG_HEAD.lock → git pull origin main --no-edit → git push
  - ⚠️ 주의: 클로드가 git 명령어 직접 실행하면 lock 파일 문제 발생 가능
    → git 명령어는 반드시 사용자 CMD에서 직접 실행할 것

[오류5] git push rejected
  - 원인: 리모트에 먼저 커밋된 내용 있음
  - 해결: del .git\ORIG_HEAD.lock → git pull origin main --no-edit → git push
